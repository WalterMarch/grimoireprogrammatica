# How to use PowerShell in this VS Code Devcontainer

1. Type `pwsh` to switch to the PowerShell shell.
2. Type `/usr/bin/pwsh` to enter the PowerShell shell.
3. Types `pwsh <scriptnameandpath.ps1>` to run the named script
4. In the *Terminal* panel, click the **Launch Profile** drop-down and choose **pwsh** to open a new terminal using the PowerShell shell.
